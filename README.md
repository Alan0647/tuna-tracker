# tuna-tracker
Unifishery Enterprise
1. 權威漁獲與空間分佈數據 (RFMOs & Global Tuna Atlas) **實測漁獲	Global Tuna Atlas	1°/5° 方格歷史產量	制定年度航次規劃**
這是推估「歷史產季」與「優良漁場」最基礎的實測資料。

FIRMS Global Tuna Atlas (GTA): 由聯合國糧農組織（FAO）與五大鮪魚 RFMOs 共同維護。它彙整了全球 1950 年至今的地理參考漁獲數據（1° 或 5° 方格分佈），是推估年度洄游路徑變遷的權威來源。

五大 RFMO 公共資料庫: * WCPFC (中西太平洋): 針對正鰹、黃鰭鮪及大目鮪提供詳細的月度漁獲空間分佈圖（Task II Data）。

IOTC (印度洋): 您提及的賽席爾船籍船舶主要受此規範，其提供的長繩釣漁獲強度圖表非常詳實。

ICCAT (大西洋): 對於藍鰭鮪（BFT）的產卵場與產季研究最為透徹。

2. 即時作業趨勢監測 (Global Fishing Watch, GFW) **即時監控	Global Fishing Watch	船隊目前熱點、作業天數	觀察競爭對手與即時漁訊**
如果您需要了解「現在」各國漁船在哪裡聚集，這是目前全球最具公信力的開源平台。

功能: 利用衛星 AIS 訊號與機器學習，將全球超過 65,000 艘商用漁船的作業熱點視覺化。

應用: 您可以透過其 Apparent Fishing Effort 地圖，觀察特定月份全球長繩釣（Longline）或圍網（Purse Seine）船隊的移動趨勢，進而反推魚群目前的地理位置。

3. 海洋環境與洄游預測模型 (SEAPODYM & Copernicus) **科學預測	SEAPODYM / CLS	環境連動、洄游模擬	氣候變遷（如聖嬰現象）下漁場偏移預判**
鮪魚是高度依賴海洋物理環境（溫度、含氧量、洋流）的物種。

SEAPODYM 模型: 這是目前國際公認最頂尖的鮪魚空間動態模型。它結合了海洋生物地球化學與物理數據，能模擬不同年齡層魚群（幼魚、成魚）在不同季節的棲息地偏好。

Copernicus Marine Service: 提供免費的近實時（Near Real-Time）海洋數據，如海表溫度 (SST)、葉綠素濃度 (Chlorophyll) 及海流方向。

專業技巧： 透過觀察「溫度鋒面（Fronts）」的移動，可預判大目鮪或長鰭鮪的季節性聚集點。

4. 產業綜論與庫存狀態 (ISSF) **永續概況	ISSF Stock Tool	全球各洋區魚種健康度	高層投資與採購決策參考**
ISSF (International Seafood Sustainability Foundation): * 推薦資料: 每年固定更新的 "Status of the World Fisheries for Tuna" 報告。

價值: 該報告不僅有產量統計，更包含針對 23 個主要商業鮪魚群（Stocks）的科學評估趨勢，能幫助您判斷特定魚種在特定洋區的產出潛力。

身為台灣遠洋漁業一員，以下本土資料庫與日本資料比對具有極高價值：
